Android SDK安装

概述

在Windows下利用Eclipse开发Android程序：

​ JDK 安装

​ Android SDK 安装

​ 配置Eclipse

JDK安装

这边推荐安装jdk8，目前最新的长期支持版本是jdk11，但是在安装Android SDK时，可能会出问题。
可以去https://repo.huaweicloud.com/java/jdk/下载，镜像比较快
安装好之后记得安装的目录，去环境变量里配置好。

Android SDK

开发Android程序要用的，官网貌似被墙了，
http://tools.android-studio.org/index.php/sdk中下载tools资源
下载安装版，可以测出是否安装了JDK，（jdk11被显示版本低，就很奇怪，换成jdk8就好了）
安装好之后打开SDK Manager，安装默认的包就可以，记得Extras里选上 最后一个 Intel x86 Emulator···这个；官网安装，会有点慢，也有可能安装失败，如果出现，可以再搜一下教程，手动安装。

AVD：Android 虚拟机
这里会提示你配置SDK的系统环境变量。
如果提示缺少kernel-qemu。可以将你创建的avd里选择D:\Android\android-sdk\system-images\android-29对应的目录下去将kernel-ranchu-64改名。

Eclipse ADT

adt：是一个插件，可以提前下载好包
开启eclipse，Help - Install New Software，安装好adt之后，Window - Reference - Android ，至此差不多了。
开启AVD Manager，如果显示黑屏，可以将Android SDK中改为5.1.1版本（可以在SDK Manager中去下载），版本过高可能会无法显示。
